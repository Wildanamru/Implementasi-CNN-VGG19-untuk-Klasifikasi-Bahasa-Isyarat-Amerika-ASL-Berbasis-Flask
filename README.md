# Implementasi CNN VGG19 untuk Klasifikasi Bahasa Isyarat Amerika (ASL) Berbasis Flask

Proyek ini mengimplementasikan model Convolutional Neural Network (CNN) menggunakan arsitektur VGG19 untuk mengklasifikasikan Bahasa Isyarat Amerika (ASL). Dengan memanfaatkan dataset berukuran besar yang terdiri dari 87.000 gambar isyarat tangan yang diperoleh dari [![Kaggle](https://img.shields.io/badge/Kaggle-ASL%20Alphabet-20BEFF?style=for-the-badge&logo=kaggle&logoColor=white)](https://www.kaggle.com/datasets/grassknoted/asl-alphabet), model ini dilatih untuk mengenali huruf dan simbol ASL dengan akurasi tinggi. Flask digunakan sebagai framework web untuk menyediakan antarmuka yang memungkinkan pengguna mengunggah gambar dan mendapatkan hasil klasifikasi secara real-time. Solusi ini dirancang untuk mendukung komunikasi yang lebih inklusif bagi penyandang disabilitas pendengaran
